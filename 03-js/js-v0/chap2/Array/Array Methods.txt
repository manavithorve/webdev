*Array Methods*
-------------------------------------------------------------------------------------------------------------------------------------------------------------
1.sort() = sorting an array
_____________________________________________________________________________________________________________________________
2.length() = length of an array  
_____________________________________________________________________________________________________________________________
3.slice(x,y)= new array from existing ,x=index of first element ,y=index of last element 
_____________________________________________________________________________________________________________________________
4.concat()= concatenation of elements seperated by comma 
_____________________________________________________________________________________________________________________________
5.join('Symbol')=same as concate except you can specify symbol
_____________________________________________________________________________________________________________________________
6.shift()=removes and returns first element of the array
_____________________________________________________________________________________________________________________________
7.push()=add element at last
_____________________________________________________________________________________________________________________________
8.pop()=remove element from last
_____________________________________________________________________________________________________________________________
9.reverse()=reverse the elements of the array
