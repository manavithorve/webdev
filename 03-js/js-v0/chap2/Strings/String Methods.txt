-------------------------------------------------------------------------------------------------------------------------------------------------------------
1. charAt(n) = Returns character at n position

3x. length=length of string

6.subStr(x,y)= creates the substring ,x=first position ,y=number of characters

8.num.toString()=Converts number to Integer String 
_____________________________________________________________________________________________________________________________

_____________________________________________________________________________________________________________________________
10. toLowerCase=Converts into Lower Case
_____________________________________________________________________________________________________________________________
11. letter.charCodeAt() = Returns the unicode of the letter

12. String.fromCharCode(code)=Returns the character of code
